# Backlink Acquisition System

This folder contains the ethical backlink acquisition pack for Somken Jobs.

Files:

- `seo-audit-summary.md` - live SEO readiness audit and fixes before scaling outreach.
- `backlink_prospects.csv` - prospecting database and outreach tracker.
- `outreach_templates.md` - manual outreach templates by audience.
- `backlinkable_content_ideas.md` - resource ideas that can earn legitimate links.
- `backlink_acquisition_roadmap.md` - 6-12 month rollout plan.

Rules:

- Do not buy links.
- Do not use fake accounts, PBNs, spam comments, or automated backlink tools.
- Do not mass-send outreach without review.
- Do not ask for exact-match anchors.
- Do not request links where there is no useful reason for the prospect's audience.
